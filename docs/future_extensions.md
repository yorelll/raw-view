# raw-view 深层次功能扩展建议

## 1. 多帧与序列能力

- 原生支持 RAW/YUV 多帧时间轴播放、逐帧缓存与快速跳转
- 支持批量导入序列文件并统一参数解析
- 增加帧间差分、运动热图与异常帧检测

## 2. 图像质量分析能力

- 增加直方图、伪彩、过曝欠曝区域标记
- 加入 SNR、对比度、清晰度等基础指标统计
- 支持两帧/两图对比（滑动、闪烁、差值）

## 3. ISP 调试辅助

- 提供基础 demosaic、白平衡、gamma、去噪的可调试链路
- 引入 Bayer 通道视图（R/Gr/Gb/B）与通道统计
- 支持保存调试参数预设并一键复现

## 4. 转换与任务自动化

- 增加批量转换队列、失败重试与任务日志
- 支持输出模板（按日期/格式/分辨率自动命名）
- 提供命令行模式，便于接入 CI 或自动化脚本

## 5. 工程协作能力

- 参数工程文件（项目配置）导入导出
- 最近任务、最近文件、团队共享配置
- 一键导出问题复现包（输入+参数+版本信息）

## 6. 平台与发布能力

- 提供跨平台打包流水线（Windows/macOS/Linux）
- 增加自动更新机制与版本回滚能力
- 完善崩溃日志采集与用户反馈通道
